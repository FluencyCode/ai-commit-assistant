# AI Commit Assistant 🤖

AI 驱动的 Git 提交助手，支持 IntelliJ 和 VSCode 双插件。

## ✨ 功能特性

- 🤖 **AI 自动生成 commit message** - 基于代码变更智能分析
- 🌍 **中英文双支持** - 自动识别语言偏好
- 📝 **规范检查** - 符合 Conventional Commits 规范
- 💻 **双插件支持** - IntelliJ IDEA + VSCode
- 🔌 **一键提交** - IDE 内完成所有操作

## 🛠️ 技术栈

| 模块 | 技术 |
|------|------|
| IntelliJ 插件 | Java + Kotlin |
| VSCode 插件 | TypeScript |
| AI 服务 | Python + FastAPI |
| 大模型 | 通义千问/Kimi |

## 📦 项目结构

```
ai-commit-assistant/
├── intellij-plugin/    # IntelliJ 插件
├── vscode-plugin/      # VSCode 插件
├── ai-service/         # AI 后端服务
└── docs/              # 文档
```
